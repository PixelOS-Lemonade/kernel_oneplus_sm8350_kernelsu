# OnePlus 9 KernelSU with KernelSU builder

> [!IMPORTANT]  
> KernelSU has officially [dropped non-gki supports on `v1.0`](https://github.com/tiann/KernelSU/releases/tag/v1.0.0) and breaks compability of manager at [`v1.0.2`](https://github.com/tiann/KernelSU/releases/tag/v1.0.2). Kernel `v0.9.5` and manager `v1.0.1` is the last version that supports non-gki devices. For now, KernelSU is pinned to `v0.9.5`. Make sure you always download and install [manager `v1.0.1`](https://github.com/tiann/KernelSU/releases/tag/v1.0.1)!
